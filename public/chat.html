First launch

Welcome to Raspi Chat

Before you jump in, here are the useful links and the best way to install the chat as a web app on your phone.

Android

For a more reliable PWA, use Chrome on Android. Open the chat, tap the browser menu, then choose Add to Home screen or Install app.

iPhone / iPad

On iOS, open the chat in Safari, tap Share, then Add to Home Screen. That way the chat opens like a standalone app.
